# News_Classification

## Getting Started

### Using Virtual Environment

- create virtual env

    ```bash
    $ python -m venv venv
    ```
- run virtual env
    - Windows
    ```bash
    $ venv\Scripts\activate.bat
    ```
    - Mac / Linux
    ```bash
    $ source venv/bin/activate
    ```
- exit virtual env
    ```bash 
    $ deactivate 
    ```
